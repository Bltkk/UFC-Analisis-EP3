# UFC Fighters - Analisis Exploratorio (EP3)

**Asignatura:** Programacion para la Ciencia de Datos  
**Evaluacion:** Parcial 3  
**Dataset:** UFC Fighters Stats (4.496 peleadores)

## Pregunta de analisis

Que caracteristicas fisicas y de estilo distinguen a los peleadores de UFC mas exitosos?

## Contenido

- `UFC_Analisis_EP33_COLAB.ipynb` - Notebook con el analisis completo
- `ufc_fighters_stats_merged.csv` - Dataset de peleadores UFC

## Visualizaciones (Plotly)

1. **Grafico de barras** - Tasa de victorias por postura (stance)
2. **Grafico de lineas** - Alcance promedio segun peso del peleador
3. **Grafico de dispersion** - Volumen vs. precision de golpeo
4. **Grafico de barras** - Tipos de victoria (KO, sumision, decision)
5. **Grafico de dispersion** - Golpes conectados vs. absorbidos

## Principales hallazgos

- La postura **Switch** tiene la mayor tasa de victorias (~74%)
- Relacion creciente y lineal entre peso y alcance
- No existe un estilo de golpeo dominante unico
- El KO/TKO es la forma de victoria mas comun (~21%)
